Task Tracker Site Plan

Overview

Purpose

The purpose of this site is to create a persistent method of tracking tasks online. Users will be able to create new tasks, mark tasks as complete, delete tasks, and view their list of tasks across multiple sessions (on the same device). Ideally this would be user specific through creation of an account, but for simplicity's sake and realistic scope, device specificity can be achieved through the use of Javascript's localstorage to demonstrate.

Audience

Anyone who has trouble keeping track of things they need to do.

Dynamic elements

Javascript will be used to bring up a form window for creating new tasks will overlay upon selecting the option to create a new task, adding the task to the display by formatting the input data into the proper HTML template. Similarly, Javascript will be used to remove HTML and localstorage data for tasks being deleted, and to update the display and localstorage data for tasks being marked as complete. The localstorage feature of Javascript will be used to persist the task data across sessions on the same device.
